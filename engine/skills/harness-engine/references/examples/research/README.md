# Research Reference Example Pack

This directory is a reference-only example pack that supplements the `research` task adapter.

Purpose:

- Shows the level of direct examples and validation rigor a research harness should have.
- Helps concretely recall essential rules like source management, research breadth, and Claim-to-Evidence separation.

Rules:

- This pack is not portable core to be copied as-is to new projects.
- It does not replace the normative contract of `references/adapters/research.md` and the project contract packet.
- The adapter itself is valid even without the example pack.

Minimum composition:

- `README.md`
- `ANTI_GOOD_REFERENCE.md`
- `VALIDATION_REFERENCE.md`
