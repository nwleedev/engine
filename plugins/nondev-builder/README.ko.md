# nondev-builder

시장 조사, 주식 분석, PRD 작성, 피치덱 등 비개발 전문 작업을 위한 Claude Code 플러그인입니다.
도메인 독립적인 방법론 주입, 병렬 웹 리서치, 독립적 결과물 평가를 통해 환각 없는 전문 작업을 지원합니다.

## 기능

- **`/nondev-setup [task-name|설명]`** — 6개 항목 브레인스토밍 루프를 통해 도메인 요구사항을 수집하고,
  병렬 웹 검색으로 최신 베스트 프랙티스를 수집하여 도메인별 세 가지 아티팩트를 생성합니다:
  - `.claude/nondev/<task-name>/skill.md` — 도메인 방법론 (B 레이어)
  - `.claude/nondev/<task-name>/rubric.md` — 평가 루브릭 (C 레이어)
  - `.claude/commands/<task-name>.md` — 작업 워크플로우 슬래시 커맨드 (A+B+C)
- **`/<task-name> [목표]`** — 방법론 주입 → 3개 병렬 리서치 서브에이전트 → 종합 → 독립 평가 서브에이전트
- **`/nondev-sync <task-name>`** — 사용자 편집 섹션 보존, 재생성 가능한 섹션만 업데이트
- **SessionStart 훅** — 세션 시작 시 설정된 도메인 목록 주입 (index.json 1회 읽기, O(1))
- **UserPromptSubmit 훅** — 프롬프트 키워드 매칭으로 관련 도메인 커맨드 제안
- **Stop 훅** — 매 턴 이후 루브릭 기반 평가 리마인더 주입

## 핵심 설계 원칙

- **하드코딩 금지**: 플러그인 파일에 도메인 이름, 키워드, 루브릭 항목 없음
- **확장 가능한 훅**: 세션 시작/프롬프트 매칭은 index.json 1파일만 읽음 (도메인 수 무관)
- **투 월드 모델**: 플러그인 파일(버전 관리, 도메인 독립) vs. 생성 파일(프로젝트 범위, 팀 공유 가능)
- **출처 강제**: 모든 수치는 외부 출처 인용 필수
- **독립 평가**: 평가 서브에이전트는 격리된 컨텍스트에서 실행 (자기 검토 편향 방지)

## 시작하기

```bash
/nondev-setup market-research
# 또는 자연어로:
/nondev-setup "주식 시장 단기 투자를 위한 종목 분석이 필요합니다"
```

설정 후:
```bash
/market-research "국내 SaaS 시장 규모 및 경쟁 현황 분석"
```

## 생성 파일 위치

```
<프로젝트>/
└── .claude/
    ├── nondev/
    │   ├── index.json                  # 도메인 매칭 인덱스 (O(1) 읽기)
    │   └── <task-name>/
    │       ├── skill.md                # 도메인 방법론
    │       └── rubric.md               # 평가 루브릭
    └── commands/
        └── <task-name>.md              # 작업 워크플로우 슬래시 커맨드
```
