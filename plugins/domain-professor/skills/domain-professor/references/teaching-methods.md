# Teaching Methods Reference

## Feynman Technique
복잡한 개념을 아이에게 설명하듯 단순화. 설명 중 막히는 부분이 이해의 공백.

## Scaffolding
기존 지식 위에 새 개념을 쌓는 방식. 항상 알고 있는 것에서 시작.

## Worked Examples
이론 전에 예시 먼저. 구체적 → 추상적 순서.

## Analogy
새로운 개념을 친숙한 것과 연결. "X는 Y와 같다, 단 Z가 다르다" 형식이 효과적.

## Prerequisite Linking
개념 간 의존 관계를 frontmatter와 파일 내 링크로 명시. 학습 경로 자동 구성.
