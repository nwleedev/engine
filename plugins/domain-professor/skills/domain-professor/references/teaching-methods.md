# Teaching Methods Reference

## Feynman Technique
Simplify a complex concept as if explaining it to a child. Wherever you get stuck is where the understanding gap lies.

## Scaffolding
Build new concepts on top of existing knowledge. Always start from what the learner already knows.

## Worked Examples
Show examples before theory. Move from concrete to abstract.

## Analogy
Connect new concepts to familiar ones. The format "X is like Y, except Z" is especially effective.

## Prerequisite Linking
Make dependencies between concepts explicit via frontmatter and in-file links. This enables automatic learning path construction.
